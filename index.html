How to run this. Move left → right through the five steps. Click any text to edit it (Enter saves, Esc cancels). Use the ◯ dot to flag a cell "discuss this". Use the ◑ swatch to recolour text (cyan / amber / red / green / grey / black) so you can mark something live. Add or remove rows & teams with ＋ and ×. Everything saves automatically; Undo reverses the last delete or reset.
